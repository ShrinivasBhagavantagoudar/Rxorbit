# RxOrbit – From Doctor’s Notes to Patient Clarity

RxOrbit is an AI-powered healthcare assistant that analyzes **handwritten medical prescriptions** and **printed diagnostic reports** to generate **structured, easy-to-understand summaries** for patients.

The system uses **OCR and AI-based contextual analysis** to extract medicine details, dosage instructions, and diagnostic values. It delivers results through a **Telegram bot** in **text, PDF, and audio formats**, supporting **English and Kannada** languages.

### Key Features
- Handwritten prescription recognition
- Diagnostic report interpretation
- Structured medical summaries
- PDF and audio (TTS) output
- Telegram-based interface
- Privacy-preserving processing

### Tech Stack
Python, PaddleOCR, Tesseract OCR, PyTorch, Transformers, Telegram Bot API, ReportLab, gTTS
